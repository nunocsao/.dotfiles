This is the quick guide for the "dataStructures" Python3 module. It provides
abstract data types (ADTs) for usage within scripts. It can be imported like
this:

>>> import dataStructures as ds

From here, you can assign your ADT of choice to a variable, like so:

>>> <VAR_NAME> = ds.<ADT_NAME>

Version 1.0 currently supports the following for <ADT_NAME>:
- Stack();
- Queue();
- Deque().

The currently available ADTs support simple methods, such as:
- Add/Remove items (of any type!!!);
- Check if the ADT is empty or not;
- Get the current size of the ADT.

Future upgrades may include improvements for the available ADTs, or support
for new ones. Unexpected errors or bugs can be reported to the email
address provided above.
